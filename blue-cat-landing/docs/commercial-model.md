# Modelo comercial

- **Blue Cat PYME:** licencia perpetua, posicionamiento estándar/listo para usar.
- **Blue Cat Enterprise:** licencia perpetua, evaluación de automatización, multi-sucursal e integraciones.
- **Blue Cat Cloud Sync:** servicio mensual opcional, separado de la licencia.
- **Actualizaciones:** 12 meses incluidos; después, el cliente conserva la última versión obtenida y puede renovar versiones futuras.
- **Pago inicial:** transferencia bancaria con revisión manual.

No se publican precios ni límites de usuarios, cajas, dispositivos o sucursales hasta que estén aprobados. También están pendientes razón social contractual, impuestos, devoluciones, soporte y SLA.
