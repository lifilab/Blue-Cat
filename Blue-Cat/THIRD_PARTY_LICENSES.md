# Inventario inicial de terceros

| Componente | Uso | Licencia/estado |
|---|---|---|
| PHP 8.3 | Runtime servidor | Verificar licencia PHP y términos de redistribución al empaquetar |
| MySQL o MariaDB | Base local | Decisión pendiente; revisar obligaciones de redistribución |
| Font Awesome | Iconos web | Confirmar versión y licencia de los recursos distribuidos |
| Navegador/PWA | Cliente | No se redistribuye navegador en la primera Beta |
| GitHub Actions | CI/release | Solo desarrollo y publicación |

Antes del instalador se debe fijar versión, origen, checksum, licencia y obligación de aviso de cada binario incorporado. El SBOM de cada release complementa, pero no reemplaza, esta revisión legal.
